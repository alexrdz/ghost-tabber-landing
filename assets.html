📸 How to export these perfect assets:

  1. Right-click anywhere on the specific image box you want and select Inspect.
  2. In the Chrome DevTools Elements panel, ensure the <div class="asset..."> element is highlighted.
  3. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac) to open the Command Menu.
  4. Type Capture node screenshot and hit Enter.
  5. Boom! Chrome will instantly download a perfectly cropped, 24-bit PNG with no alpha channel (solid background) at the exact pixel dimensions required by the Web Store.
