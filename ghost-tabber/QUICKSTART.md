# 👻 Ghost Tabber - Quick Start (2 minutes)

## Install

1. Open Chrome/Edge
2. Go to `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the `ghost-tabber` folder
6. Done! 🎉

## Configure

1. Click the 👻 Ghost Tabber icon in your toolbar
2. Make sure **Auto-close tabs** is ON (toggle should be blue)
3. Set **Close tabs after:** to your preference (default: 30 minutes)
4. Leave checkboxes enabled:
   - ✅ Keep pinned tabs open
   - ✅ Keep tabs with audio open

## Test (Optional)

**Quick 5-minute test:**

1. Set timeout to **5 minutes** in the popup
2. Open 3-4 random tabs (e.g., Wikipedia, GitHub)
3. Switch to a different tab and wait 6 minutes
4. The test tabs should automatically close
5. Click 👻 icon → **📚 Open Archive** to see them
6. Click **↗️ Restore** on any tab to reopen it

## Use

That's it! The extension now:
- ✅ Tracks which tabs you use
- ✅ Automatically closes inactive tabs
- ✅ Saves them in a searchable archive
- ✅ Lets you restore them anytime

**View archived tabs:**
Click 👻 icon → **📚 Open Archive**

**Search archive:**
Type in the search box on the archive page

**Restore a tab:**
Click **↗️ Restore** next to any archived tab

## Tips

- Set a realistic timeout (30 min - 2 hours recommended)
- Pin important tabs so they never close
- Periodically clear old tabs from the archive
- Use search to find that tab you can't remember

## Troubleshooting

**Tabs not closing?**
- Check extension is enabled (toggle in popup)
- Verify timeout has elapsed
- Check if tabs are pinned/active/playing audio

**Archive empty?**
- Wait for full timeout duration
- Check stats in popup (shows archived count)

**Extension not loading?**
- Check `chrome://extensions` for errors
- Verify all files are present
- Reload the extension
